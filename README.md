# Auth-app

[App link](https://developerbello.github.io/auth-app/)

`Auth-app` is a form validation application.

To run the App, do the following,

```sh
git clone https://github.com/developerbello/auth-app.git
```

```sh
cd auth-app/
```

```sh
  yarn start
```

## main features

- Form fields
- Submit button
- form input validation
- etc.
